直线的生成算法_DDA

DDA是数字微分分析式（digital differential analyzer）的缩写
设直线的起点为x1,y1x_1, y_1x1​,y1​,终点为x2,y2x_2, y_2x2​,y2​,则斜率为
k=y2−y1x2−x1=ΔyΔxk = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\Delta y}{\Delta x}k=x2​−x1​y2​−y1​​=ΔxΔy​

直线中的每一个点坐标都可以是有前一个点坐标变换一个增量Δx,Δy\Delta x, \Delta yΔx,Δy 而得到，即表示为递归式
xi+1=xi+Δxyi+1=yi+Δxx_{i + 1} = x_i + \Delta x \\y_{i + 1} = y_i + \Delta xxi+1​=xi​+Δxyi+1​=yi​+Δx

由于递归式的初值是直线的起点x1,y1x_1, y_1x1​,y1​,这样就可以使用加法来生成一条直线，具体方法为： 按照直线从x1,y1x_1, y_1x1​,y1​到x2,y2x_2, y_2x2​,y2​的方向不同，分为8个象限，如下图

不考虑和坐标轴平行的场景

象限Δx\Delta xΔxΔy\Delta yΔy
1a象限Δx=1\Delta x = 1Δx=1Δy=k\Delta y = kΔy=k
1b象限Δx=1k\Delta x = \frac{1}{k}Δx=k1​Δy=1\Delta y = 1Δy=1
2a象限Δx=−1\Delta x = -1Δx=−1Δy=−k\Delta y = -kΔy=−k
2b象限Δx=1k\Delta x = \frac{1}{k}Δx=k1​Δy=1\Delta y = 1Δy=1
3a象限Δx=−1\Delta x = -1Δx=−1Δy=−k\Delta y = -kΔy=−k
3b象限Δx=−1k\Delta x = -\frac{1}{k}Δx=−k1​Δy=−1\Delta y = -1Δy=−1
4a象限Δx=1\Delta x = 1Δx=1Δy=k\Delta y = kΔy=k
4b象限Δx=−1k\Delta x = -\frac{1}{k}Δx=−k1​Δy=−1\Delta y = -1Δy=−1

  2. 自增的符号与x2−x1x_2 - x_1x2​−x1​， y2−y1y_2 - y_1y2​−y1​的方向相同

使用DDA算法，每生成一条直线需要做两次触发，每画线中的一点做两次加法操作，因此，使用DDA生成的直线的速度是相当快的
