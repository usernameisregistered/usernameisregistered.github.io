01 直线的生成算法 BresenHam

设直线的从起点(x1,y1)(x_1, y_1)(x1​,y1​)到终点(x2,y2)(x_2,y_2)(x2​,y2​),直线可以表示方程y=kx+by = kx + by=kx+b

假定直线的点是位于第一象限，并且斜率是大于 0 小于 1 的，这这种情况下，当直线光栅化时，xxx每次都是自增 1,而相应的yyy的增量是小于 1 的，为了光栅化，当xix_ixi​增加 1 时，y 只能是yiy_iyi​或者yi+1y_i + 1yi​+1两个位置之一

yiy_iyi​或者yi+1y_i + 1yi​+1的位置选择的原则是有精确值yyy与yiy_iyi​和yi+1y_i + 1yi​+1的距离d1d_1d1​和d2d_2d2​的大小而定的，计算公式如下

y=k(xi+1)+bd1=y−yid2=yi+1−yy = k(x_i + 1) + b \\ d_1 = y - y_i \\ d_2 = y_i + 1 - yy=k(xi​+1)+bd1​=y−yi​d2​=yi​+1−y

如果d1−d2>0d_1 - d_2 > 0d1​−d2​>0,则选择yi+1yi + 1yi+1, 否则选择yiy_iyi​,因此算法的关键在与简便的求出d1−d2d_1 - d_2d1​−d2​的符号，
d1−d2=2y−2yi−1=2m(xi+1)−2yi+2b−1(1)d_1 - d_2 = 2y - 2y_i -1 = 2m(x_i + 1) - 2y_i + 2b -1 \tag{1}d1​−d2​=2y−2yi​−1=2m(xi​+1)−2yi​+2b−1(1)
m=y2−y1x2−x1(2)m = \frac{y_2 - y_1}{x_2 - x_1} \tag{2}m=x2​−x1​y2​−y1​​(2)
令Δx=x2−x1\Delta x =x_2 - x_1Δx=x2​−x1​, Δy=y2−y1\Delta y =y_2 - y_1Δy=y2​−y1​
将2式带入1式
Δx(d1−d2)=2xiΔy−2yiΔx+2Δy+Δx(2b−1)(3)\Delta x(d_1 - d_2) = 2x_i\Delta y - 2y_i\Delta x + 2\Delta y + \Delta x(2b-1) \tag{3}Δx(d1​−d2​)=2xi​Δy−2yi​Δx+2Δy+Δx(2b−1)(3)

由于已经假定直线的 2 个端点都是在第一象限，并且斜率是大于 0 小于 1 的,所以Δx\Delta xΔx大于 0,因此Δx(d1−d2)\Delta x(d_1 - d_2)Δx(d1​−d2​)依然可以用作符号判断

令Pi=Δx(d1−d2)P_i = \Delta x(d_1 - d_2)Pi​=Δx(d1​−d2​)
则 3 式表示为
Pi=2xiΔy−2yiΔx+2Δy+Δx(2b−1)(4)P_i = 2x_i\Delta y - 2y_i\Delta x + 2\Delta y + \Delta x(2b-1) \tag{4}Pi​=2xi​Δy−2yi​Δx+2Δy+Δx(2b−1)(4)

Pi+1=2xi+1Δy−2yi+1Δx+2Δy+Δx(2b−1)P_{i+1} = 2x_{i+1}\Delta y - 2y_{i+1}\Delta x + 2\Delta y + \Delta x(2b-1)Pi+1​=2xi+1​Δy−2yi+1​Δx+2Δy+Δx(2b−1)
Pi+1−Pi=2Δy−2Δx(yi+1−yi)(5)P_{i+1} - P_i = 2\Delta y - 2\Delta x(y_{i + 1} - y_i) \tag{5}Pi+1​−Pi​=2Δy−2Δx(yi+1​−yi​)(5)

综合所述,针对上述条件下的直线 BresenHam 算法思想如下

  1. 绘制第一个点 计算Δx,Δy,P0=2Δy−1\Delta x, \Delta y, P_0 = 2\Delta y - 1Δx,Δy,P0​=2Δy−1
  2. 求下个位置xi+1=xi+1x_{i+1} = x_i + 1xi+1​=xi​+1若Pi>0P_i > 0Pi​>0则取yi+1y_{i+1}yi+1​否则yiy_iyi​
  3. 绘制
  4. 求下一个误差,若Pi>0P_i > 0Pi​>0则取Pi+1=Pi+2Δy−2ΔxP_i+ 1 = P_i + 2\Delta y - 2 \Delta xPi​+1=Pi​+2Δy−2Δx否则Pi+1=Pi+2ΔyP_i+ 1 = P_i + 2\Delta yPi​+1=Pi​+2Δy
  5. 判断是否需要终止，否则执行2、3、4
