语法正则

markdown语法参考

标题

标题语法：

  1. 在单词或短语的前面添加对应个数的 # 即可,在 # 后面至少要有一个空格/(#{1,6})(?=\s|$)/
  2. 在单词或短语的前面添加对应标题的HTML标签 例如 <h1> 这种暂时不做处理
  3. 在#或者HTML标题标签前面最多添加3个空格 / {0,3}/
// /(.*)(?:\n+|$)匹配末尾或者换行前的任意字符，并且换行只匹配但不记住该匹配项
const heading = /^ {0,3}(#{1,6})(?=\s|$)(.*)(?:\n+|$)/;

水平线

要创建水平线，请单独在一行上使用三个或更多的*, -, _, 符号之间可以存在空格或者制表符
前面最多添加3个空格 / {0,3}/

const hr = /^ {0,3}((?:-[\t ]*){3,}|(?:_[ \t]*){3,}|(?:\*[ \t]*){3,})(?:\n+|$)/;

代码块

代码块通常缩进四个空格或一个制表符。当它们在列表中时，将它们缩进八个空格或两个制表符。

const blockCode = /^( {4}[^\n]+(?:\n(?: *(?:\n|$))*)?)+/;

清单

要创建有序列表，请在订单项中添加数字和句点。数字不必按数字顺序排列，但列表应以数字开头
要创建无序列表，请在订单项前添加破折号（-），星号（*）或加号（+）。缩进一个或多个项目以创建嵌套列表

const bullet = /(?:[*+-]|\d{1,9}[.)])/;

围栏代码块

可以在代码块之前和之后的行上使用三个刻度线（`）或三个波浪号（~） 创建代码块 /(`{3,}(?=[^`\n]*(?:\n|$))|~{3,})/
前面最多添加3个空格 / {0,3}/

const space = / {0,3}/
/**
 * `{3,} 匹配3个```
 * ?=[^`\n]* 后面不为换行符和`的任意多个字符
 * (?:\n|$) 匹配结尾或者换行符
 * ~{3,})匹配3个~
 */
const begin = /(`{3,}(?=[^`\n]*(?:\n|$))|~{3,})/
/**
 * ([^\n]*) 匹配单行
 * (?:\n|$) 匹配空格但是不记录
 * (?:|([\s\S]*?)(?:\n|$))
 */
const content = /([^\n]*)(?:\n|$)(?:|([\s\S]*?)(?:\n|$))/
/**
 *  {0,3} 前面最多添加3个空格
 * \1 反向引用
 * [~`]* 匹配连续的零个或者多个~`
 *  *(?=\n|$)| 匹配连续的零个或者多个空格
 * $ 匹配结尾
 */
const end = /(?: {0,3}\1[~`]* *(?=\n|$)|$)/
// fences = /^space begin content end/
const fences = /^ {0,3}(`{3,}(?=[^`\n]*(?:\n|$))|~{3,})([^\n]*)(?:\n|$)(?:|([\s\S]*?)(?:\n|$))(?: {0,3}\1[~`]* *(?=\n|$)|$)/;
